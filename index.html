Mad Libs

In this project, I created a Mad Libs game using JavaScript. The game prompts the user to enter words, which are then used to fill in the blanks of a story. I used the getText function to retrieve the user's input and the setText function to display the completed story. I also used string concatenation to combine the user's input with the story template.

This project helped me to understandhow to use JavaScript to create interactive web applications. I learned how to use functions to retrieve and display user input, and how to manipulate strings. I also learned how to use event listeners with console.log to respond to user actions, such as clicking a button.
